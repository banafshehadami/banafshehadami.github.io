# Banafsheh Academic Homepage

This repository contains your GitHub Pages academic profile site.

## Files

- `index.html` page content (about, news, publications, links)
- `style.css` page styles and responsive layout
- `script.js` dark mode toggle and dynamic year

## Publish (GitHub Pages)

1. Create a repository named `banafshehadami.github.io`
2. Put these files in the repository root
3. Push to GitHub
4. Open `https://banafshehadami.github.io`

## Quick PowerShell Commands

```powershell
git init
git add .
git commit -m "Initial academic homepage"
git branch -M main
git remote add origin https://github.com/banafshehadami/banafshehadami.github.io.git
git push -u origin main
```
